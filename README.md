# student-performance-analysis
Project Overview  This project focuses on analyzing student academic performance using Microsoft Excel. It helps in understanding students’ strengths and weaknesses by evaluating their marks in different subjects.  The analysis is performed on a dataset of 100 students with marks in Mathematics, English, and Science. 
#Objectives
Analyze student performance using Excel
Calculate total marks and percentage
Identify top-performing and weak students
Determine pass and fail rate
Compare subject-wise performance
Dataset Description



#The dataset contains the following columns:
Name
Math Marks
English Marks
Science Marks
Total Marks
Percentage
Result (Pass/Fail)
#Tools & Technologies
Microsoft Excel
Basic Excel Functions:
SUM
AVERAGE
IF
COUNTIF
Charts (Column Chart, Pie Chart)


#Key Features
Automatic calculation of total marks and percentage
Pass/Fail classification using IF function
Subject-wise average calculation
Identification of top and bottom performers
Data visualization using charts


#Analysis Performed
Class average for each subject
Pass and fail count
Pass percentage calculation
Top 5 and bottom 5 students
Subject comparison (Math, English, Science)

#Conclusion
This project demonstrates how basic Excel tools can be used to perform effective data analysis. It is simple, efficient, and useful for educational purposes.
